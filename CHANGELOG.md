# Changelog

本檔自 repo 根目錄 `VERSION` 投影產生（`node scripts/genVersion.mjs`）；請勿手改，改沿革請編輯 `VERSION`。
版號釘選於 PR merge（依變更型別 bump VERSION），release 與版號解耦；本檔收全部變更，遊戲 About 只投影 playerVisible 筆。

## 0.54.0 — 2026-06-22 _(internal)_
- feat (#245): 管理編輯工具（Luminara Editor）新增「場景設定」頁籤：可檢視各場景名稱／角色／背景並檢視與手動編修場景對話（打工任務、生活聊天題庫），另提供依提示詞 AI 自動生成對話（無金鑰時降級為輸出提示詞供外部模型貼回）；屬本機 dev 作者工具、不影響公開遊戲

## 0.53.1 — 2026-06-22
