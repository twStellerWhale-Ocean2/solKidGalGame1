拖拉與鍵盤操作

在預覽圖上直接拖拉所選的框：中央 ✛ 方塊移動、四邊四角 8 點縮放（非等比）。選「無選擇」可隱藏所有框，純看衣物。也可用方向鍵微調位置（1px；Shift＝10px），或直接在下方數值欄輸入。

框數值 left / top / right / bottom

數值、拖曳與預覽三向同步；還原＝回到未調整前（未套用的變更才可還原）。

°

聲音管理

為各角色類型（依性別與性格）指定要使用的瀏覽器語音，讓不同人物聲線可被真正區分。

指定與繼承規則

每個性別有「預設」一列，性格類型未指定時沿用同性別預設；留在 Auto 的類型由系統依語言自動挑選。此設定為整台裝置共用（device-wide），與遊戲共用同一份指定（localStorage）；公開 GitHub Pages 不提供此入口。

試聽範例句

在左側每列按「▶ 試聽」，會以該類型目前選定的語音唸這句範例（Auto／繼承時以系統解析之同性別語音試聽）。

指定即時寫入裝置共用儲存，不需另存檔；重新整理遊戲即生效。本頁不需 dev server，但入口僅於本機 dev 環境出現。
