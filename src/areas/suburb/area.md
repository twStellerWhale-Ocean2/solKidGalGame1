# Suburb Area Pack

Suburb is the zoomed-in resource production region from the upper-right countryside of the Kingdom panorama.

## Gameplay

- `Kingdom Road`: portal marker back to Kingdom.
- `Mine`, `Logging Camp`, `Fishing Shore`, `Pasture`, `Farm`, `Mill`, and `Village Home`: Help scenes using Cambridge A1 Movers-style resource vocabulary.
- Each Help scene owns five local questions in `manifest.js`.

## Asset Status

- Uses `assets/areas/suburb-map.png` and `assets/scenes/suburb-scenes-atlas.png`, generated as finished bitmap storybook game art for this region.
