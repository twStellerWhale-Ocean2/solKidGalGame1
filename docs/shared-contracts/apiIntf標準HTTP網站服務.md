# I. 主旨目的

定義 HTTP 網站與網頁檔案服務的標準 API，用於以 URL 取得 HTML、CSS、JavaScript、圖片與其他靜態資源。

# II. 參考準備

* HTTP 定義 request、response、method、status code、header 與 content negotiation 等基本語意。
* URI 定義網路資源定位格式。
* HTTP Semantics：https://www.rfc-editor.org/rfc/rfc9110
* URI Generic Syntax：https://www.rfc-editor.org/rfc/rfc3986

# III. 內容程序

* 直接參考 HTTP 與 URI 的官方標準文件。
* 標準協定，由社群軟體及AI直接編寫。設計師不須參與細部內容。
