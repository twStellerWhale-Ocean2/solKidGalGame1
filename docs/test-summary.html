剩餘風險

Web Speech API 的聲音品質仍受 OS、瀏覽器、安裝 voice 與使用者裝置喇叭影響。Code 階段已處理工程可控項與診斷，但「首字是否對兒童足夠清楚」仍需 opr 實機聽測確認。
