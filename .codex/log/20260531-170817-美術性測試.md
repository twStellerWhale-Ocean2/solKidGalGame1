## 固定檢查清單

每張 manifest 截圖必須逐項檢查：

- HUD / stat / header 是否完全在 viewport 內，沒有超出邊界或造成水平捲動。
- 文字是否裁切、擠壓、溢出容器，按鈕是否被遮擋或太貼邊。
- 背景、角色、NPC、道具與 UI 是否像同一空間與同一套視覺語言。
- 角色比例、站位、裁切、腳底接地、陰影 / 底座是否自然。
- 對話框、Shop / Wardrobe panel、設定 panel 是否壓壞舞台構圖。
- Scene entry 是否只顯示場景、NPC / 角色與 action choices。
- `Area Map -> Scene -> Action Choices -> Detail Panel` 層級是否可由截圖辨識。
- 使用者提供的 Castle 圖是否確實成為 Castle map 背景。
