# III. 內容程序

## 判定

- 主畫面焦點回到 Princess Lumi、Wardrobe、Travel Map，符合「選地方、聽一句、拿獎勵、換裝」主循環。
- 齒輪是低干擾入口；孩子不需要理解 Save MD / Load MD 就能開始玩。
- Diary 仍可作為完成任務後的回顧頁，不從主畫面搶焦點。
- Save / Load 被歸為 family save page，降低誤觸與沉浸破壞。

## 殘留風險

- Settings 仍包含 OpenAI Help Key 表單，視覺上比 Diary 更偏系統；因為已收進 overlay，本輪接受。
- 後續若要更兒童化，可把 Settings 再拆成「家長頁」與「聲音/單字等級」兩層。

# IV. 備註紀錄

好玩性測試通過；本輪改動提升主畫面遊戲感，未改動核心任務難度與獎勵節奏。
