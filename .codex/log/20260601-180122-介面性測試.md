# 20260601-180122 介面性測試

## 目標

驗證地圖拖曳 / 縮放、marker focus、鍵盤操作與返回路徑一致，且 HUD / bottom area nav 不被地圖 viewport 影響。

## 檢查項目

- 手機觸控拖曳 / pinch。
- 滑鼠或等效 pointer drag。
- 方向鍵 / W/S / Enter / Space / 數字鍵 smoke。
- Back / Leave / system overlay 返回路徑。

## 結果

- 已確認 CSS 層面：
  - Kingdom `.map-stage > img` 不再使用 `--map-pan-x/y` transform。
  - Kingdom `.map-stage > img` 使用 `left/top/width/height` 對齊 `syncAreaMapStyles()` 產出的 display / offset。
  - Castle `.castle-stage > img` 使用同一組 display / offset 變數。
  - Castle `.castle-stage` 補上 `touch-action: none` 與 `user-select: none`，避免瀏覽器原生手勢干擾地圖 pointer flow。
- 未完成：手機觸控拖曳 / pinch、滑鼠 drag、鍵盤操作、Back / Leave / overlay 返回的實際操作驗證。
- 結論：介面 contract 靜態檢查通過；實際操作測試因 Browser gate 阻塞未完成。
