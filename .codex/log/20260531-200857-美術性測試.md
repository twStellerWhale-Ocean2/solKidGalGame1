## 10 點視覺審查

### Must Fix

無。沒有阻擋本輪發布的視覺問題。

### Should Fix

1. 使用者手動把地圖拖到邊緣時，公主 token 可能部分貼近 viewport 邊界；這是可接受的拖曳結果，後續可考慮加入「回到公主」快捷操作。
2. selftest JSON 畫面是工程測試輸出，不是正式 gameplay UI；若未來要分享給非工程使用者，可另外做測試報告頁。

### Accept

1. Castle / Kingdom 可見標題已移除，畫面更像可探索地圖。
2. Hidden heading / aria label 保留，視覺簡化沒有犧牲可及性入口。
3. Castle 公主 token 清楚提供目前位置線索。
4. Kingdom 公主 token 明顯高於 marker，不會被地點 icon 遮住。
5. 公主 token 不阻擋 marker 點擊，互動 affordance 保持清楚。
6. 所有 area nav 按鈕維持綠色，視覺規則一致。
7. Active area 使用公主小頭像，手機上比額外文字更容易掃描。
8. 地圖 zoom / drag 後 HUD 與 nav 不跟著縮放，手機操作穩定。
9. marker focus ring 在手機截圖中可辨識。
10. Scene entry 與 return path 的畫面切換沒有出現重疊、截斷或不可讀文字。

美術性測試通過。每個 manifest row 都有手機直向截圖與檢查結論，沒有 Must Fix。
