# Shop / Wardrobe 主舞台試穿修訂紀錄

## 主旨

本輪作業模式為「修正目前功能」。目標是讓 Shop / Wardrobe 的衣物、鞋子與配件試穿直接套在 ADV 上方既有 Princess Lumi 主舞台，不再產生獨立小紙娃娃或小試穿畫面。

## README 同步

- 已先同步 `README.md` 的 Room、Shop 與變更紀錄。
- 新規格：進入 Shop / Wardrobe detail panel 時不自動試穿第一件；玩家點選商品後才暫時試穿。
- 新規格：暫時試穿不寫入 outfit state；只有 `BUY` 或 `Equip` 才正式保存。

## 修訂內容

- `src/main.js`：新增 transient try-on rendering，讓 `data-doll="adv"` 的主舞台 Lumi 使用暫時 outfit。
- `src/main.js`：Shop / Wardrobe detail 初始不設定 `shopPreviewItemId`，分類切換、Back、Leave、close ADV 會清除未確認 preview。
- `src/main.js`：移除 `renderShopPreview()` 產生的 `.shop-feature` 與 `.shop-preview-doll` 小舞台。
- `src/flow/adv-controls.js`：支援 disabled ADV option，未選商品時顯示不可點的 `Pick a treasure`。
- `styles/shop.css`、`styles/mobile.css`、`styles/adv.css`：調整無小試穿面板後的 Shop / Wardrobe layout，並加入主舞台試穿狀態視覺提示。
