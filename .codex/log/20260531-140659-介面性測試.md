# III. 內容程序

## 通過項目

- 預設 Room 第一眼是 Princess Lumi、房間與 Wardrobe，不再是底部系統控制列。
- 齒輪位於狀態區右上，符合「設定入口」直覺，不佔主流程。
- Diary / Settings / Save Load 以書本 overlay 呈現，符合 README 的「遊戲內道具」方向。
- 三個 menu tab 觸控尺寸約 38px 高；Save / Load 按鈕約 46px 高。
- Back 按鈕可關閉 overlay；Escape 也可關閉。
- `Save MD` / `Load MD` 只在 Save / Load page 可見。

## 殘留觀察

- Settings 內容較長，手機上需要在書本內捲動；此狀態可接受，因為它是家長/系統頁，不是主遊玩回合。
- Map mobile 的 `Back to Room` toolbar button 為既有隱藏狀態，返回依靠 Room hotspot；已記錄為非本輪範圍。
