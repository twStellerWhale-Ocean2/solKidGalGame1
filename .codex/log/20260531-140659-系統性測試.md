## Menu state

- `#diary`、`#settings`、`#save` 會開啟 system menu，而不是切換成主遊戲 view。
- 關閉 menu 後 hash 會回到目前主 view，例如 `#home`。
- Escape 可關閉 menu。
- 關閉後 focus 回到齒輪按鈕。
